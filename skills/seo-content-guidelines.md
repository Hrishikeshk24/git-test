# SEO Content Guidelines Skill

## Instructions

Apply these SEO best practices to maximize search visibility and SERP ranking.

### Keyword Strategy

1. **Primary Keyword Placement**
   - Include in H1 title (once, naturally)
   - Use in first 100 words (within first 2-3 sentences)
   - Repeat 2-3 times in body (naturally, not forced)
   - Include in meta description (155-160 chars)
   - One mention per H2 section (3-4 H2 sections)

2. **Keyword Variations**
   - Include related terms (LSI keywords)
   - Use synonyms and long-tail variations
   - Cover intent variations (how-to, comparison, definition)

3. **Meta Description Template**
   ```
   [Hook/benefit]. [Specific outcome]. [CTA]
   155-160 characters exactly.
   Include primary keyword once.
   ```

### Content Structure

**H1 Title:**
- One per page
- Include primary keyword
- Make it compelling (this is your headline)
- Example: "How to [achieve outcome] + [specific benefit] in [timeframe]"

**H2 Sections (3-4 sections):**
- Descriptive, keyword-aware headings
- Structure: Problem → Solution → Example
- 300-500 words per section
- Include one data point per section

**Paragraphs:**
- Keep short (3-4 sentences max)
- Use transition words (However, Therefore, Additionally)
- Bold key terms (Ctrl+B in most editors)

### E-E-A-T Signals

Make your content demonstrate:

**Expertise:**
- Cite data, research, or studies
- Reference industry standards
- Include specific methodologies

**Experience:**
- Share case studies or real results
- Reference customer outcomes
- Include "we've seen" or "we've worked with"

**Authority:**
- Link to authoritative sources
- Cite recognized experts
- Reference tenure/track record with evidence

**Trustworthiness:**
- Be transparent about limitations
- Acknowledge counterarguments
- Disclose any biases or sponsorships
- Show author credentials briefly

### Internal Linking Strategy

- Link to 2-3 related content pieces
- Use anchor text that includes keywords
- Link to content that answers follow-up questions
- Place links naturally in paragraphs, not forced

### Content Length & Read Time

- **Target length**: 2000-2500 words for blog posts
- **Target read time**: 7-10 minutes
- **Sections**: 3-4 H2 sections minimum
- **Break readability**: Subheadings every 300-400 words

### Image Strategy

- Include 1-2 relevant images
- Use descriptive alt text with keywords
- Optimize file size (< 100KB per image)
- Add image captions with context

### Final Checklist

- [ ] Primary keyword in H1, first 100 words, meta description
- [ ] 3-4 H2 sections with keyword variations
- [ ] Short paragraphs (3-4 sentences max)
- [ ] E-E-A-T signals throughout (at least 3 different types)
- [ ] 2-3 internal links with keyword anchor text
- [ ] 2000+ words, 7+ minute read time
- [ ] 1-2 images with alt text
- [ ] Meta description 155-160 chars with keyword
- [ ] No keyword stuffing or forced tone changes
- [ ] Provides unique perspective or data not found elsewhere
